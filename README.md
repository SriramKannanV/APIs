# DummyAPI Demo Project

A responsive React-based API demo project built using DummyAPI to fetch and display dynamic data. This project demonstrates API integration, component-based architecture, state management, loading handling, and clean UI development practices.

## Features

* Fetch data from DummyAPI
* Responsive UI design
* Reusable React components
* Loading & error handling
* Clean and modern interface
* API-based dynamic rendering

## Tech Stack

* React JS
* JavaScript
* CSS
* DummyAPI

## Installation

```bash
git clone <your-github-repo-link>
cd <project-folder>
npm install
npm run dev
```

## Learning Outcomes

* Working with REST APIs
* Handling asynchronous requests
* Managing component states
* Building responsive frontend applications
